bsync options

If "remote" is selected, all bookmarks from this computer will be uploaded to AWS S3.

If "home" is selected, all bookmarks will be downloaded from AWS S3 to this computer. In that case, you will need to choose a target folder where the remote bookmarks will be saved to. Note: everything in this folder is cleared out and replaced with remote bookmarks when a sync operation occurs. Please ensure that the home computer only has one folder with the target name you provide.

AWS S3 configuration:

AWS credentials are stored unencrypted in Chrome's local storage and are never uploaded to the cloud. The access key entered here should have read/write access to the bucket.
